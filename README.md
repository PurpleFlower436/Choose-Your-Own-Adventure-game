# Choose Your Own Adventure Game: Ancient Rome

This is a text-based Choose Your Own Adventure game where users travel through ancient Rome and face different challenges based on the choices they make.

## Features
- User accounts so players can have their information saved
- Branching story paths where players pick different choices in each scenario
- Save function that allows players to save progress and continue later


## Tech Stack
- HTML
- CSS
- JavaScript


## What I Implemented
- Used HTML and CSS for the UI.
- Designed a flowchart mapping story branches and user choices to ensure consistent, logical paths through the game.
